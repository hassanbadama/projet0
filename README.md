# projet0

AHP Decision Tool
Outil d'aide à la décision basé sur la méthode AHP (Analytic Hierarchy Process)

Qu'est-ce que c'est ?
Une application web qui vous aide à choisir la meilleure option quand vous avez :

Plusieurs critères (prix, qualité, confort, etc.)

Plusieurs alternatives (voiture A, voiture B, voiture C)

Démarrage rapide
Prérequis
Java 21

Maven (ou utilisez le fichier mvnw inclus)

Installation
bash
# 1. Cloner le projet
git clone https://github.com/hassanbadama/projet0


# 2. Lancer l'application
./mvnw spring-boot:run
Accès
Ouvrez votre navigateur : http://localhost:8080

 Comment utiliser ?
En 5 étapes simples :
Étape	Action	Exemple
1	Ajoutez vos critères	Prix, Qualité, Design
2	Ajoutez vos alternatives	Voiture A, Voiture B
3	Notez chaque alternative (0-10)	Prix: Voiture A=8, Voiture B=5
4	Comparez les critères entre eux	Prix vs Qualité = 3 (Prix plus important)
5	Obtenez la recommandation	 Meilleure option: Voiture A
 Exemple concret
Problème : Choisir une voiture

Critères : Prix, Sécurité, Confort

Alternatives : Voiture A, Voiture B
